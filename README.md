### Descrição
o projeto consiste na composoição de uma tabela que conta com capitais brasileiras e a distancia entre essas capitais. As capitais constantes nesse projeto são:
 
* Campo Grande
* Cuiabá
* Curitiba
* Florianopolis
* Fortaleza
* Goiania
* Joao Pessoa
* Macapá
